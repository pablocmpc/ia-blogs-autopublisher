📌 ¿Qué hace este sistema?

El Auto-Publisher IA es un sistema que genera y publica artículos de blog de forma completamente automática en 3 sitios web WordPress, 7 veces al día, 365 días al año, sin intervención humana.

El sistema usa inteligencia artificial para escribir artículos SEO de alta calidad en español, busca imágenes relevantes automáticamente, y los publica directamente en WordPress. Todo esto de forma gratuita usando GitHub Actions como servidor.

🔧 ¿Cómo funciona? — Paso a paso

1
⏰ GitHub dispara el cron
A las 07:00, 09:30, 12:00, 14:30, 17:00, 19:30 y 21:30 hora española, GitHub Actions lanza automáticamente el sistema.
2
🔑 Selecciona keyword
Para cada web, elige aleatoriamente una keyword de su lista (75-200 temas por web). Nunca repite la misma dos veces.
3
🤖 IA genera el artículo
Envía la keyword a Groq AI (gratis). En 30 segundos devuelve un artículo completo de 1.500-2.200 palabras con estructura SEO.
4
🖼️ Busca imagen
Busca una foto relevante y libre de derechos en Pexels API (gratis). La descarga y la sube al WordPress de cada web.
5
📝 Publica en WordPress
Crea el post directamente via WordPress REST API con título SEO, URL amigable, categoría, etiquetas y meta descripción.
6
📌 Pin en Pinterest
Si Pinterest está configurado, crea automáticamente un pin enlazando al artículo. Más tráfico sin esfuerzo extra.

⚙️ ¿Qué necesita el usuario para operarlo?

Nada. El sistema es 100% autónomo. Sin embargo, para mejores resultados se recomienda:

  • Revisar el dashboard 1 vez/semana para confirmar que todo está verde
  • Añadir nuevas keywords al banco cuando se agoten (cada ~2-3 meses)
  • Revisar Google Search Console mensualmente para ver qué términos posicionan
  • Cuando AdSense genere ingresos, optimizar los artículos que más tráfico traigan

🚀 Escalabilidad

El mismo sistema se puede ampliar fácilmente:

  • Añadir más webs WordPress (solo hay que añadir credenciales en config.json)
  • Aumentar a 3 artículos por turno (de 21 a 63/día) cambiando un número
  • Conectar Instagram y Facebook para publicar en redes sociales automáticamente
  • Añadir envío automático de newsletters con los artículos de la semana
  • Vender el sistema como servicio a otras empresas
Modelo SaaS: Este mismo sistema configurado para el negocio de un cliente tiene un valor de mercado de 500-1.500€ de setup + 200-400€/mes de mantenimiento. Con 10 clientes: 2.000-4.000€/mes recurrentes.
